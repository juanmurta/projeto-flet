# Projeto Chat com Flet

## Descrição
Este é um aplicativo de chat simples desenvolvido com Flet, uma biblioteca Python para criação de interfaces de usuário interativas para web, desktop e dispositivos móveis. O aplicativo permite que os usuários entrem em uma sala de chat com um nome de usuário e troquem mensagens em tempo real.

## Funcionalidades
- Interface de usuário simples e intuitiva
- Sistema de login com nome de usuário
- Notificações quando novos usuários entram no chat
- Troca de mensagens em tempo real
- Visualização web acessível via navegador

## Requisitos
- Python 3.7 ou superior
- Biblioteca Flet

## Instalação

### 1. Clone o repositório
```bash
git clone https://github.com/seu-usuario/projeto-flet.git
cd projeto-flet
```

### 2. Instale as dependências
```bash
pip install flet
```

## Como usar

### 1. Execute o aplicativo
```bash
python main.py
```

### 2. Acesse o chat
Após executar o aplicativo, ele será aberto automaticamente no seu navegador padrão no endereço `http://localhost:4567`.

### 3. Entre no chat
- Clique no botão "Iniciar Chat"
- Digite seu nome de usuário
- Clique em "Entrar"

### 4. Envie mensagens
- Digite sua mensagem no campo de texto
- Pressione Enter ou clique no botão "Enviar"

## Estrutura do projeto
- `main.py`: Contém todo o código do aplicativo, incluindo a interface do usuário e a lógica de comunicação
